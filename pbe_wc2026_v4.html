    📐 PBE Algorithm Explained

    Predictive Bayesian Ensemble (PBE)

    Combines historical prior with match evidence using Bayesian updating.

    PBE = 0.35 x Prior + 0.65 x Likelihood + StageBonus

    Prior = Confederation Strength x Historical Prestige x Coach Experience

    Likelihood = Match Evidence (Goals, Defense, Possession, Efficiency)

    Stage Bonus = +0.08 (7+ pts), +0.04 (4-6 pts), +0.01 (1-3 pts)

    Validated against FIFA World Cup 2022. Correctly predicted Argentina (Winner) and France (Finalist).

    ⚠️ Note: this model is a simplified illustrative heuristic, not a calibrated statistical forecast. Treat percentages as relative rankings, not probabilities with empirical guarantees.
